# Atividade 2 — Poções & Soluções
Projeto 2 de Web no ICMC. Frontend em React, backend em Node.js + Express + SQLite em memória.

## Como rodar

### 1. Clonar o repositório
```bash
git clone https://github.com/GlaucoFleuryCM/atividade2_Web.git
cd atividade2_Web
```

### 2. Backend
```bash
cd back
npm install
node server.js
```
Servidor sobe em http://localhost:3000

### 3. Frontend
Abra um novo terminal:
```bash
cd front
npm install
npm start
```
Site abre em http://localhost:3001
